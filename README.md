# **What is it Worth?** 
Machine learning models created for the Machine Learning Zoomcamp. A live version of this project can be found at [whatisitworth.co.za](https://www.whatisitworth.co.za)

Note: this project is specific to South Africa and its car market. 

## **1. Problem Description**
I was looking for a car to buy. It isn't always easy to know if a price is good or not at first glance, and it can be a hassle trying to search through a ton of different listings on various car sites, trying to compare different options. 

I wanted to make a tool that lets somebody get a quick and easy appraisal so that they have an idea of what kind of price is reasonable. To make this as easy as possible, users only need to submit the make, model, variant, year, mileage, and province to get an initial evaluation of a vehicle. 



## **2. Prerequisites**


### 2.1. Installing Docker on Debian-based linux

```bash
# Remove the conflicting Ubuntu repository
sudo rm /etc/apt/sources.list.d/docker.sources

# Remove old Docker GPG key
sudo rm /usr/share/keyrings/docker-archive-keyring.gpg

# Add Docker's official GPG key for Debian
sudo install -m 0755 -d /etc/apt/keyrings
sudo curl -fsSL https://download.docker.com/linux/debian/gpg -o /etc/apt/keyrings/docker.asc
sudo chmod a+r /etc/apt/keyrings/docker.asc

# Add the correct Debian repository
echo \
  "deb [arch=$(dpkg --print-architecture) signed-by=/etc/apt/keyrings/docker.asc] https://download.docker.com/linux/debian \
  $(. /etc/os-release && echo "$VERSION_CODENAME") stable" | \
  sudo tee /etc/apt/sources.list.d/docker.list > /dev/null

# Update and install
sudo apt update
sudo apt install docker-ce docker-ce-cli containerd.io docker-buildx-plugin docker-compose-plugin
```

### 2.2. Commands for Docker:
- To build the docker container:
```bash
docker build -t whatisitworth .
```

-  Running the docker container:
```bash
docker run -t -p 80:80 whatisitworth:latest
```

- Running the docker container in a detached state
```bash
docker run -d -p 80:80 whatisitworth:latest
```

-  To see docker processes currently running:
```bash
docker ps
```
Which should output something like:
